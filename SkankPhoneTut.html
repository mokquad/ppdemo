SkankPhone Tutorial

Follow the steps below to install SkankPhone on iPod touch 1g / iPhone 2g:

  1. Downgrade your iPod touch 1g / iPhone 2g to iPhone OS 1.1.1 - 1.1.5. I prefer to use 1.1.2.
  2. Restore the device using iTunes 7.5.
  3. Jailbreak the device using ZiPhone.
  4. Download the file system (a link will be provided below).
  5. Copy all files from /System/Library/PrivateFrameworks/ [dump] and paste them into the same directory on your device.
  6. Navigate to /AppleInternal/Applications/SkankPhone.app [dump] and copy the folder to /Applications on your device.
  7. ENJOY!
